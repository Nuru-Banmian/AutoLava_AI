# Triage 标签

这些技能使用五种规范的 triage 角色。下表将这些角色映射到本仓库问题跟踪器中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库中的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估此问题 |
| `needs-info` | `needs-info` | 等待报告者提供更多信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交由 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时（例如“应用 AFK-ready triage 标签”），应使用本表中对应的标签字符串。

如果实际使用的标签词汇发生变化，请编辑右侧“本仓库中的标签”列。
