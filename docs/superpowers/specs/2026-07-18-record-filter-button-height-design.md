# 营业记录筛选按钮高度统一设计

## 目标

让营业记录页顶部筛选行的操作按钮与日期输入框保持相同的 44px 高度，消除同一行控件的视觉不一致。

## 范围

仅修改 `RecordFilters` 中的四个按钮：本月、上月、自定义、导出当前范围。

- 四个按钮均增加 `min-h-11`（44px）高度契约。
- 保留当前横向内边距、文字大小、选中态、禁用态、导出逻辑与换行行为。
- 开始日期与结束日期标签保持当前位置和当前样式。
- 桌面与手机端均使用相同高度；筛选行仍可换行，不引入横向溢出。

不修改日期输入组件、日期范围逻辑、接口、导出请求、其它页面按钮或全局按钮组件。

## 验收与测试

- 通过 `RecordFilters` 测试断言四个按钮都包含 `min-h-11`。
- 现有预设选择、有效性校验和导出行为测试继续通过。
